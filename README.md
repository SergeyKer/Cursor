# Сайт-портфолио

Современный и адаптивный сайт-портфолио с красивым дизайном и плавными анимациями.

## Особенности

- ✨ Современный и чистый дизайн
- 📱 Полностью адаптивный (работает на всех устройствах)
- 🎨 Плавные анимации и переходы
- 🚀 Быстрая загрузка
- ♿ Доступность и удобство использования
- 🎯 SEO-оптимизированная структура

## Структура проекта

```
портфолио/
├── index.html      # Основная HTML структура
├── styles.css      # Стили и дизайн
├── script.js       # JavaScript функциональность
└── README.md       # Документация
```

## Секции сайта

1. **Главная** - Приветственная секция с кратким описанием
2. **О себе** - Информация о вас и ваша статистика
3. **Проекты** - Портфолио ваших работ
4. **Навыки** - Ваши технические навыки с прогресс-барами
5. **Контакты** - Форма обратной связи и контактная информация

## Как использовать

1. Откройте `index.html` в браузере
2. Настройте контент под себя:
   - Замените тексты в HTML файле
   - Обновите ссылки на ваши проекты
   - Измените контактную информацию
   - Добавьте свои навыки и проекты

## Настройка

### Изменение цветовой схемы

Откройте `styles.css` и измените переменные в секции `:root`:

```css
:root {
    --primary-color: #6366f1;    /* Основной цвет */
    --secondary-color: #8b5cf6;  /* Вторичный цвет */
    /* ... другие цвета */
}
```

### Добавление проектов

Добавьте новые карточки проектов в секцию `#projects`:

```html
<div class="project-card">
    <div class="project-image">
        <div class="project-placeholder">Ваш проект</div>
    </div>
    <div class="project-content">
        <h3 class="project-title">Название проекта</h3>
        <p class="project-description">Описание проекта</p>
        <div class="project-tags">
            <span class="tag">Технология</span>
        </div>
        <div class="project-links">
            <a href="#" class="project-link">Демо</a>
            <a href="#" class="project-link">Код</a>
        </div>
    </div>
</div>
```

### Настройка навыков

Измените прогресс-бары в секции `#skills`, изменив значение `width`:

```html
<div class="skill-progress" style="width: 90%"></div>
```

### Обновление контактной информации

Найдите секцию `#contact` и обновите:
- Email адрес
- Ссылки на социальные сети
- Текст приветствия

## Развертывание

### GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы в репозиторий
3. Перейдите в Settings → Pages
4. Выберите ветку `main` и папку `/root`
5. Ваш сайт будет доступен по адресу `https://username.github.io/repository-name`

### Netlify

1. Зарегистрируйтесь на [Netlify](https://www.netlify.com)
2. Перетащите папку проекта в область загрузки
3. Сайт будет автоматически развернут

### Vercel

1. Зарегистрируйтесь на [Vercel](https://vercel.com)
2. Импортируйте репозиторий
3. Нажмите Deploy
