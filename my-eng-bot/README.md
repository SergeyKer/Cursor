# My Eng Bot — мой друг по английскому

Сайт для практики английского: диалог с ИИ (OpenRouter) и режим «Тренировка перевода». Озвучка ответов по кнопке, голосовой ввод по кнопке микрофона, настройки в выезжающем меню слева.

## Требования

- Node.js 18+
- Ключ API [OpenRouter](https://openrouter.ai/) (бесплатный лимит: 50 запросов/день для моделей `:free`)

## Установка и запуск

```bash
npm install
cp .env.example .env.local
# В .env.local укажи OPENROUTER_API_KEY=твой_ключ
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000).

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `OPENROUTER_API_KEY` | Ключ API OpenRouter (обязателен для чата и счётчика запросов) |

## Деплой на Vercel (с нуля)

Репозиторий содержит папку **my-eng-bot** — при деплое укажи её как **Root Directory**.

### Способ 1: Через сайт (рекомендуется)

1. Открой: **[Import my-eng-bot на Vercel](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FSergeyKer%2FCursor&root-directory=my-eng-bot)**.
2. Войди в Vercel (через GitHub или email).
3. Проверь:
   - **Root Directory** = **`my-eng-bot`** (обязательно, без пробела);
   - **Framework Preset** = Next.js (подставится сам).
4. Нажми **Deploy**. Дождись окончания сборки.
5. Добавь переменную окружения:
   - В проекте: **Settings** → **Environment Variables**;
   - **Name:** `OPENROUTER_API_KEY`, **Value:** твой ключ с [OpenRouter](https://openrouter.ai/);
   - Сохрани, затем **Deployments** → у последнего деплоя **⋯** → **Redeploy**.

Готово: сайт будет по адресу вида `https://твой-проект.vercel.app`.

### Способ 2: Через CLI

```bash
cd my-eng-bot
npx vercel login   # один раз: откроется браузер для входа
npx vercel         # деплой (превью)
npx vercel --prod  # деплой в прод
```

Переменную `OPENROUTER_API_KEY` потом добавь в [vercel.com](https://vercel.com) → проект → **Settings** → **Environment Variables** и сделай **Redeploy**.

## Функции

- **Диалог**: ИИ начинает первым, ответы только текстом; озвучка по кнопке «Озвучить» у каждого сообщения ИИ.
- **Тренировка перевода**: ИИ выдаёт предложения на русском (общие / вопросительные / отрицательные / смешанные), пользователь вводит или диктует перевод; обратная связь с блоком исправлений.
- **Настройки** (меню слева): режим, тип предложений, тема, уровень (Starter–C2), грамматическое время, выбор голоса, счётчик запросов OpenRouter.
- **Микрофон**: распознавание английской речи (Web Speech API), текст подставляется в поле ввода.
- **История** хранится в браузере (localStorage). При смене режима чат очищается и начинается новая беседа.
