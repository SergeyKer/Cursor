# Принципы перевода с русского на английский

Краткая методичка для преподавателя и для настройки ИИ. Жёстких списков «только так» нет: решение по **контексту** — сильная привязанность / глубина чувства против нейтрального «нравится».

## Love и like

| Принцип | Содержание |
|--------|------------|
| Разница по силе | **Like** — слабее: предпочтение, симпатия, привычка, «мне нравится». **Love** — сильнее: глубокая привязанность, сильная эмоция; в разговорной речи также усиление вроде «обожать» (еда, хобби), когда по смыслу уместно. |
| Не калькировать «люблю» | Русское **люблю** не переводить автоматически как *love*. Смотреть смысл: часто достаточно *like*, *enjoy*, *be fond of*. |
| Контекст важнее правил | Один и тот же русский глагол может дать *I like reading* или *I love reading* в зависимости от того, насколько сильно говорящий выражает чувство в тексте. |
| Справочно (носители) | Учебные источники описывают **разницу интенсивности**, а не «запрет» на *love* в определённых темах. Например: [Cambridge — hate, like, love, prefer](https://dictionary.cambridge.org/grammar/british-grammar/hate-like-love-and-prefer). Для питомцев *I love my dog* — обычная коллокация при сильной привязанности. |

**Эвристика для обучения:** «это про глубокую привязанность / сильное чувство или про обычное предпочтение?» — первое ближе к *love*, второе к *like*.

## Другие пары

- **Would like** — вежливое «хочу» (*I'd like to…*), не путать с эмоциональным *love*.
- **I'd love to** — сильнее, чем *I'd like to* («с удовольствием / очень хочу»).
- При сомнении — выбирать по тону текста; допустимо держать в уме оба варианта.

## Связь с API

По умолчанию [`/api/translate`](../app/api/translate/route.ts) переводит **английский → русский** (кнопка «Перевод» у ответа ассистента).

Для перевода **русский → английский** передайте в теле запроса `"direction": "ru_to_en"` вместе с полем `text` на русском. Подробности — в комментариях к маршруту.
