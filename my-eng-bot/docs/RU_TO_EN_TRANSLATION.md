# Принципы перевода с русского на английский

Краткая методичка для преподавателя и для настройки ИИ. Для пары **like / love** в продукте действует **одна** согласованная политика (см. также [`lib/likeLoveTutorPrompt.ts`](../lib/likeLoveTutorPrompt.ts) и системные промпты чата / [`/api/translate`](../app/api/translate/route.ts) для `ru_to_en`).

## Like и love

| Принцип | Содержание |
|--------|------------|
| Сила | **Like** — нейтральное предпочтение, привычка, «мне нравится» (слабее). **Love** — сильная привязанность, восторг, «люблю / обожаю» (сильнее). |
| Русский «нравится» | Обычно → **like** (не усиливать до *love* без смысла). |
| Русское «люблю», «обожаю» | Когда речь о чувстве, семье, близких, питомцах или явном энтузиазме → **love**; не ослаблять до *like*, если по тексту ясна сильная эмоция. |
| Эталон в упражнении | В режиме перевода в чате канонический английский задания задаёт целевое слово; ответ ученика сравнивают с русской фразой и с этим эталоном. |
| Диалог | Эталон — последняя реплика преподавателя на английском: не требовать *love*, если в вопросе *like*; ответ с *love* при том же смысле можно принять и кратко похвалить. |

**Запрещено:** менять *like* и *love* местами без семантического основания; требовать *love* при нейтральном «нравится»; требовать *like* при явном сильном «люблю» в значении чувства.

## Would like и would love

- **Would like** — вежливое «хочу» (*I'd like to…*).
- **I'd love to** — сильнее по готовности / энтузиазму, чем *I'd like to*.

## Связь с API

По умолчанию [`/api/translate`](../app/api/translate/route.ts) переводит **английский → русский** (кнопка «Перевод» у ответа ассистента).

Для перевода **русский → английский** передайте в теле запроса `"direction": "ru_to_en"` вместе с полем `text` на русском. Подробности — в комментариях к маршруту.
