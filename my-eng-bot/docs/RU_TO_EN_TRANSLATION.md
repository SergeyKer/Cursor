# Принципы перевода с русского на английский

Краткая методичка для преподавателя и настройки ИИ в режиме перевода.

## Базовые принципы

- Перевод должен быть естественным разговорным английским, без буквальных кальк.
- Сохраняйте исходный смысл, тон и намерение пользователя.
- Выбирайте понятные структуры и лексику, соответствующие уровню учащегося.
- Не добавляйте объяснений и метакомментариев в результат.

## Связь с API

По умолчанию [`/api/translate`](../app/api/translate/route.ts) переводит **английский → русский**.

Для перевода **русский → английский** передайте в теле запроса `"direction": "ru_to_en"` вместе с полем `text` на русском.
