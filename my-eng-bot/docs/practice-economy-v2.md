# Экономика практики v2

Канон реализации: план `practice_xp_plan_0556f3df`. Урок не меняется этим документом.

## Четыре режима

| Режим | Длина | XP к уровню | Особая награда | Кубок/монеты | «Не считать ошибку» |
|-------|------:|------------|----------------|--------------|---------------------|
| Эталон | 7 | 0 | проверка типа упражнения | нет | нет |
| Лёгкая | 6 | ×0.55 | разминка | нет | нет |
| Обычная | 9 | ×1 | «База» при 8/9 с первой попытки (один раз) | нет | нет |
| Челлендж | 12 | ×1.15 | 11/12 с первой попытки → 📝 | 3-й +1🪙; 5-й +2🪙; 5+золото → 🏆 | да, 1🪙, шаги 5–12 |

## Ключевые правила

- **Mastery** = только с первой попытки. Correction даёт 40% Session XP («звёзды»), но не поднимает mastery и не раздувает Global `base` (`firstTrySessionXp`).
- **Global XP:** ≥50% mastery + медаль урока; Эталон всегда 0; daily cap 70 (local day).
- **📝:** только Challenge 11/12 (+ effective forgiveness), tier>0, max 1/тема/день, cap 5. Не из fingerprint/XP-пути.
- **Монеты практики:** `coinLedger.practiceMilestones` (`{topicId}:ring3` / `ring5`). Без золота урока — pending до входа после gold.
- **COMBO** в практике — только UI, без XP.
- Кошелёк: стартовые 10 при первом создании; **нет** автопополнения 0→10 при каждой загрузке.

## Briefing / finale

- Feed: 1 bubble. Thesis в карточке: будет / уже есть / не входит.
- Finale: «с первой попытки N/M», звёзды, XP к уровню или почему нет, особая строка (📝/База/near-miss/🪙).
