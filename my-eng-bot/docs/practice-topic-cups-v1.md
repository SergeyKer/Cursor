# Кубки тем (practice topic cups v1)

## Правило

**Тема сдана** = золотая медаль 🥇 в профиле урока + **5 практик** по теме (eligible: не `reference`, результат ≥ 50%).

Маркер в UI: **🏆 ✓**, в данных: `PracticeTopicProgress.cupClaimed`.

## Флаг

`NEXT_PUBLIC_FEATURE_PRACTICE_TOPIC_CUPS_V1` — по умолчанию **включён** (opt-out: `=false`).

Не путать с `NEXT_PUBLIC_FEATURE_PRACTICE_GEMS_V1` (камни в кошелёк — отдельный будущий этап).

## Где видно пользователю

- Экран медали после урока — `cupLine` про кубок (любая медаль).
- Прогресс — «Лучший заработок», список тем, счётчик `🏆 тем: X/Y`.
- Практика — футер `🏆 n/5`, финал, popup «Тема сдана! 🏆».
- Список уроков — бейдж `🏆 n/5` / `🏆 ✓` рядом с медалью.
- Мой план — intro со ссылкой на Прогресс (карточки к кубку — v2).

## Не в v1

- Зачисление камней 💎 в `rewardsState.currencies.gems`.
- Умные рекомендации «до кубка» в «Мой план».
