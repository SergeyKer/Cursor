# Eng Bot — помощник по английскому

Сайт для практики английского: диалог с ИИ (OpenRouter) и режим «Тренировка перевода». Озвучка ответов по кнопке, голосовой ввод по кнопке микрофона, настройки в выезжающем меню слева.

## Требования

- Node.js 18+
- Ключ API [OpenRouter](https://openrouter.ai/) (бесплатный лимит: 50 запросов/день для моделей `:free`)

## Установка и запуск

```bash
npm install
cp .env.example .env.local
# В .env.local укажи OPENROUTER_API_KEY=твой_ключ
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000).

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `OPENROUTER_API_KEY` | Ключ API OpenRouter (обязателен для чата и счётчика запросов) |

## Деплой на Vercel

Репозиторий содержит папку **Eng Bot** — при деплое укажи её как корень.

### Через сайт Vercel

**Вариант А — импорт с уже заданной папкой (проще всего):**

1. Открой ссылку: [Import Cursor → Eng Bot](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2FSergeyKer%2FCursor&root-directory=Eng%20Bot).
2. Если попросит — выбери аккаунт и репозиторий `SergeyKer/Cursor`.
3. Убедись, что в поле **Root Directory** указано **`Eng Bot`** (часто подставляется из ссылки).
4. Нажми **Deploy**.

**Вариант Б — проект уже создан, нужно указать корень:**

1. Зайди в проект на [vercel.com](https://vercel.com) → выбери проект (например, **cursor**).
2. В левой колонке открой **Settings**.
3. Внутри настроек открой раздел **Build and Deployment** (или **Build & Development**).
4. Прокрути вниз до блока **Root Directory**.
5. Нажми **Edit** рядом с Root Directory, введи **`Eng Bot`**, нажми **Save**.
6. Вкладка **Deployments** → у последнего деплоя нажми **Redeploy**.

По желанию в **Settings → Environment Variables** можно добавить `OPENROUTER_API_KEY` (ключ можно вводить и в приложении в меню).

### Через CLI

```bash
cd "Eng Bot"
npx vercel
```

При первом запуске войди в аккаунт Vercel и укажи, что деплоишь текущую папку. Для продакшена: `npx vercel --prod`.

## Функции

- **Диалог**: ИИ начинает первым, ответы только текстом; озвучка по кнопке «Озвучить» у каждого сообщения ИИ.
- **Тренировка перевода**: ИИ выдаёт предложения на русском (общие / вопросительные / отрицательные / смешанные), пользователь вводит или диктует перевод; обратная связь с блоком исправлений.
- **Настройки** (меню слева): режим, тип предложений, тема, уровень (Starter–C2), грамматическое время, выбор голоса, счётчик запросов OpenRouter.
- **Микрофон**: распознавание английской речи (Web Speech API), текст подставляется в поле ввода.
- **История** хранится в браузере (localStorage). При смене режима чат очищается и начинается новая беседа.
